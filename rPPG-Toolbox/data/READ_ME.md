# Data

This folder contains the datasets used for training and evaluation.  
**No raw data is included in this repository** - datasets must be obtained separately and placed here manually.

---

## Folder Structure

```
data/
├── MMPD/
│   ├── raw/                          # Raw MMPD video data (place here after download)
│   ├── preprocessed/                 # Auto-generated by rPPG-Toolbox during preprocessing
│   └── preprocessed/DataFileLists/   # CSV file lists for 5-fold cross validation splits
│
└── UBFC-rPPG/
    ├── raw/                          # Raw UBFC-rPPG video data (place here after download)
    └── preprocessed/                 # Auto-generated by rPPG-Toolbox during preprocessing
```

---

## Obtaining the Datasets

### MMPD
MMPD (Multi-Modal Physiological Dataset) requires a formal access request.

- **Paper:** [MMPD: Multi-Domain Mobile Video Physiology Dataset](https://arxiv.org/abs/2302.03840)
- **Access:** Fill out the request form linked in the official repository - [MMPD GitHub](https://github.com/McJackTang/MMPD_rPPG_dataset)
- Once approved, place the raw data under `data/MMPD/raw/`

### UBFC-rPPG
- **Paper:** [UBFC-rPPG Dataset](https://sites.google.com/view/ybenezeth/ubfcrppg)
- **Access:** Request access via the link on the official page above
- Once downloaded, place the raw data under `data/UBFC-rPPG/raw/`

---

## Notes

- **The datasets are not included in this repository.** MMPD requires a formal written request to the authors and cannot be redistributed. UBFC-rPPG is similarly restricted to registered users only. Please obtain both datasets through the official channels listed above before running any experiments.
- The `preprocessed/` folders are generated automatically by rPPG-Toolbox when you run a config with `DO_PREPROCESS: True`- do not create them manually.
- All configs in this repo already point to these relative paths, so no path changes are needed as long as you follow the folder structure above.
